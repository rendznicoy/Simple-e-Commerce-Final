> **A responsive e-commerce platform integrated with Firebase for real-time data management.**

This project is a lightweight online shopping application that combines a custom HTML/CSS/JS frontend with the power of Firebase. Unlike static sites, this application features persistent data storage, allowing for dynamic product loading and cloud-based inventory management.

## Key Features

* **Cloud-Integrated Database:** Fetches product details and stock information in real-time using **Firebase**.
* **Dynamic Content:** Product grids are generated programmatically based on live data, ensuring the store is always up-to-date.
* **Interactive Cart:** Users can add items, manage quantities, and review their subtotal instantly.
* **Responsive Design:** A fluid interface that provides a seamless shopping experience across desktops, tablets, and mobile devices.

---

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Backend-as-a-Service:** Firebase (Realtime Database / Firestore)
